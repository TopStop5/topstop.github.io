// quickstart

Up and running in
three steps.

01
Clone the repo
Grab the source from GitHub. Python 3 and Google Chrome must be installed on your machine.
git clone https://github.com/TopStop5/novelscraper.git
02
Install dependencies
NovelScraper auto-installs missing packages on first run, but you can install them upfront:
pip install -r requirements.txt
03
Run and scrape
Launch the CLI, paste your novel URL, set your chapter range, and choose your output format. Done.
python novelscraper.py
